SIA: Smart Ingest AI 🚀
Middleware Inteligente de Abastecimiento para Puntos de Venta
SIA es un sistema diseñado para automatizar y optimizar la carga de mercadería en quioscos y mini-markets. Mediante una arquitectura de triple contingencia (Manual, OCR Local e IA), el software extrae metadatos de facturas de proveedores, normaliza los artículos y genera reportes listos para integrar con sistemas de gestión externos.

Este proyecto se desarrolla en el marco del Proyecto Integrador 2026 de la Tecnicatura Superior en Ciencia de Datos e Inteligencia Artificial del ISPC.

🩺 El Problema (Pain Point)
La carga manual de facturas de proveedores en los comercios minoristas es lenta y propensa a errores debido a:

Diversidad de formatos de comprobantes.

Complejidad impositiva (IVA, Percepciones de IIBB, Tasas).

Falta de normalización en los nombres de los productos.

SIA resuelve esto garantizando una carga eficiente con una meta de "Diferencia Cero" entre la información física y la digital.

🛠️ Tecnologías y Metodología
Lenguaje: Python 3.x

Análisis de Datos: Pandas, NumPy.

IA/OCR: Vision API / Tesseract / Agentes LLM.

Gestión: Metodología Ágil (Scrum/Kanban) mediante Trello.

Fundamentación: Basado en Data Science for Business (Provost & Fawcett).
